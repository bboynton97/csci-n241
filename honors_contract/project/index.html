What is TinyS3?

A simple Python S3 upload library. Inspired by one of our favorite packages, requests. Tinys3 is used at Smore to upload more than 1.5 million keys to S3 every month. This package is easy to set up and easy to use, making file transfers from S3 easier than ever.


How does it work?

Create a tinsyS3 connection object using your S3 Access Key and S3 Secret Key. Create a file object with the file you'd like to upload. Pass the upload file to the tinyS3 object and find it on S3. 

        import tinys3

        conn = tinys3.Connection(S3_ACCESS_KEY,S3_SECRET_KEY,tls=True)

        f = open('some_file.zip','rb')
        conn.upload('some_file.zip',f,'my_bucket')

Why use it?

Not only is TinyS3 easy to implement, but it is much more lightweight than the traditional boto package. Rather than downloading and importing countless unneccesary tools through boto, you can import one single package for all of your S3 upload and download needs.
